# JavaScript 陣列處理方法

[filter(), find(), forEach(), map(), every(), some(), reduce()]

filter() 方法會建立一個經指定之函式運算後，由原陣列中通過該函式檢驗之元素所構成的新陣列。

words.filter(word => word.length > 6);

JS:

const words = ['spray', 'limit', 'elite', 'exuberant', 'destruction', 'present'];

const result = words.filter(word => word.length > 6);

console.log(result);
// expected output: Array ["exuberant", "destruction", "present"]

ref: https://developer.mozilla.org/zh-TW/docs/Web/JavaScript/Reference/Global_Objects/Array/filter

# CheckBox 控制

延伸 v-on change :

當 Onchange Checkbox 時 this.todo.completed 變成相反









 





 <input type="checkbox" v-on:change="markComplete"/>

    <script>
        export default {
            name: "TodoItem",
            props: ["todo"],
            methods:{
                markComplete(){
                    this.todo.completed = !this.todo.completed; 
                }
            }
        }
    </script>

# 子組件中呼叫父組件

```$emit / $on```

emit 發出

是次目的是要當按Button時，把 todo.id 傳到 App.vue

Todoitems.vue -> Todos.vue -> App.vue

Todoitems.vue

<button @click="$emit('del-todo',todo.id)" class="del">x</button>

當收到TodoItem 的 del-todo 時 傳送開始

Todos.vue

<TodoItem v-bind:todo="todo" v-on:del-todo="$emit('del-todo',todo.id)"/>

當收到TodoItem 的 del-todo 時 開始 deleteTodo 的 function

App.vue

<Todos v-bind:todos="todos" v-bind:todoMsg="TodoMsg" v-on:del-todo="deleteTodo" />

    ...
        methods:{
              deleteTodo(id){
                  console.log(id)   
                this.todos = this.todos.filter(todo => todo.id !== id);
              }
        }
